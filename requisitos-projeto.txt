Precisamos de um encurtador de link que seja de uso irrestrito e ilimitado

- as urls devem ter no maximo 5 caracteres sendo de 0 a 9 e de a z 
- é necessario verificar se ja existe a url encurtada ATIVA ou seja se ela não venceu ainda
- é necessario que as urls geradas sejam salvas em um banco de dados relacional mysql com a hora da criação e data de vencimento, o id o link de redirecionamento e o hash de saida
- a url deve durar somente 3 dias a partir da data de geração
- urls vencidas redirecionam para a pagina inicial
- ao gerar a url ela deve ser copiada automaticamente para o clipboard do usuario
- para os estilos é necessario usar o boostrap 5+
- deve se postar a url e mostrar a url ao cliente via ajax
